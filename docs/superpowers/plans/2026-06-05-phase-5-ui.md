# Plan 5 — Interface — Implementation Plan

> **Statut : IMPLÉMENTÉ.** App buildée de bout en bout (`wails build` → exe 50 Mo).

**Goal :** Interface Svelte sombre et moderne, branchée aux bindings/events du backend.

## Fichiers
```
app.go            + SelectVideo (dialogue natif), bindings déjà en place
main.go           + DragAndDrop{EnableFileDrop:true}
frontend/src/App.svelte   interface complète
frontend/src/style.css    base sombre #0d0d0d
```

## Interface (App.svelte)
- **En-tête** : logo dégradé + titre + tagline.
- **Zone de dépôt** : glisser-déposer natif (`OnFileDrop`) **ou** clic → `SelectVideo` (dialogue). Affiche le nom du fichier, « Analyse des pistes… » pendant le scan.
- **Carte réglages** : contrôle segmenté **Local · GPU / Gemini · API** ; selon le moteur, liste des `.gguf` (`ListModels`) ou clé API + modèle Gemini ; langue source ; **piste de sous-titres** (les pistes image sont marquées et désactivées).
- **Actions** : `Test · 20 s`, `Démarrer`, et `Annuler` pendant un job.
- **Progression** : barre animée + compteur de lignes ; barre dédiée au **téléchargement CUDA** (Mo) au 1er lancement.
- **Bannières** succès/erreur + **journal** type console (auto-scroll).

## Câblage
- Au montage : `GetConfig` (restaure moteur/langue/clé/modèle), `ListModels`, abonnements `EventsOn` (`progress`, `log`, `download`, `done`, `error`).
- `Démarrer/Test` → `SaveConfig` puis `StartTranslation(req)`.
- `canStart` (réactif) : vidéo + piste texte sélectionnée + (modèle local | clé API).

## Validé
- Build complet OK (vite + Go), bindings générés (`SelectVideo`, modèles `mkv.Track`/`config.Config`).
- Non-régression : `go test ./...` vert.

## Reste à valider sur machine réelle (1er run)
- Téléchargement CUDA (~620 Mo) + lancement `llama-server` + inférence GPU.
- Traduction bout-en-bout d'une vraie vidéo.
- Rendu visuel de la fenêtre.
